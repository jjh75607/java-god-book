<h4>1장 프로그래밍이란 무엇인가?</h4>

<ol>
    <li>자바의 가장 작은 단위는 클래스</li>
    <li>클래스는 상태(변수)와 행동(메소드)가 있어야 한다 - 반드시는 X</li>
    <li>수학의 = 과는 다른 프로그래밍의 = </li>
    <li>한 줄이 끝날때 마다 종료를 의미하는 ;(세미콜론) </li>
</ol>
